# CardDecks

A set of classes that implement the common features
of standard decks of playing cards.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'card_decks'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install card_decks
